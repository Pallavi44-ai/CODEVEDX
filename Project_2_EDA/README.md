# Project 2 - Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Superstore Sales dataset to understand patterns, trends, correlations, and outliers.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Analysis Performed

1. Statistical Summary
2. Mean, Median, Mode
3. Standard Deviation
4. Correlation Analysis
5. Heatmap Visualization
6. Outlier Detection
7. Category and Region Analysis

## Key Findings

* Sales contain several outliers.
* Most transactions occur in lower sales ranges.
* Categories contribute differently to profit and sales.
* Correlation analysis helps understand relationships among variables.

## Outcome

EDA provided valuable insights into business performance and customer purchasing patterns.
