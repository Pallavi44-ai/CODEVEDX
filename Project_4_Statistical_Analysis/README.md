# Project 4 - Statistical Analysis using Python

## Project Overview

This project applies statistical techniques to analyze the Superstore Sales dataset. The analysis focuses on descriptive statistics, correlation analysis, hypothesis testing, and interpretation of business performance.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Statistical Methods Applied

1. Mean
2. Median
3. Mode
4. Variance
5. Standard Deviation
6. Correlation Analysis
7. Hypothesis-Based Analysis
8. Category-wise Statistical Analysis
9. Region-wise Statistical Analysis

## Key Findings

* Sales and profit exhibit measurable statistical relationships.
* Different categories show different profit distributions.
* Correlation analysis helps identify patterns among numerical variables.
* Statistical measures provide insights into business performance and variability.

## Outcome

Statistical analysis helped interpret data patterns, variability, and relationships, supporting data-driven decision-making.
