# Project 3 - Data Visualization with Python

## Project Overview

This project focuses on visualizing Superstore Sales data using Python. Various chart types were used to represent sales, profit, category performance, regional trends, and customer purchasing behavior in an easy-to-understand format.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Visualizations Created

1. Sales by Category (Bar Chart)
2. Profit by Region (Bar Chart)
3. Monthly Sales Trend (Line Chart)
4. Sales Distribution (Histogram)
5. Sales vs Profit (Scatter Plot)
6. Category-wise Sales Contribution (Pie Chart)

## Key Insights

* Technology category generated the highest sales.
* The West region contributed significantly to overall profit.
* Monthly sales showed fluctuations throughout the year.
* Most sales transactions were concentrated in lower sales ranges.
* Sales and profit generally had a positive relationship.
* Visualizations helped identify trends and business opportunities.

## Outcome

The project successfully transformed raw sales data into meaningful visual insights. The charts provide a clear understanding of sales performance, profitability, and business trends, supporting data-driven decision-making.
